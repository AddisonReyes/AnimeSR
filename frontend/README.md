# Frontend

Aplicacion en `Next.js + TypeScript` para consumir el backend y mostrar recomendaciones en vivo.

## Flujo de la UI

- Barra de busqueda para encontrar un anime.
- Etiquetas clicables por genero.
- Recomendaciones renderizadas en tarjetas.
- Modal con informacion completa del anime seleccionado.

## Variables

- `NEXT_PUBLIC_API_BASE_URL`
  - Por defecto: `http://127.0.0.1:8000`

## Ejecutar

```bash
npm install
npm run dev
```
